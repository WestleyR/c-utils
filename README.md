# C-Utils - Simple C Utils For Your C Code

<br>

## Install

Install by [clib](https://github.com/clib/clib):

## Example

```
$ clib install WestleyR/c-utils
```

```c

#include <stdio.h>

#include "c-utils/c-utils.h"

int main() {

    // No example here?...

    return(0);
}
```

Then add `CFLAGS += -I deps` to your Makefile.
